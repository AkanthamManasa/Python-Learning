## Key Concepts

**Variable** — a named box that holds a value. 

**f-string** — mix variables into text.

**List** — ordered collection. 

**Arithematic Operations** -to perform mathematical operations.

## List Functions Cheatsheet
| Function  | What it does   | SQL equivalent |
|-----------|----------------|----------------|
| len()     | count of items | COUNT(*)       |
| sum()     | total          | SUM()          |
| max()     | largest        | MAX()          |
| min()     | smallest       | MIN()          |
| .append() | add to end     | INSERT         |
